# Interview Scheduler

An easy-to-use app that allows you to book, edit and cancel interviews. Built in React, as a part of Lighthouse Labs bootcamp.

## Dependencies:

axios,
classnames,
normalize.css,
react,
react-dom,
react-scripts

## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```
